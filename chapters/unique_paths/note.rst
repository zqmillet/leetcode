不同路径
========

题目描述
--------

一个机器人位于一个 ``m * n`` 网格的左上角 (起始点在下图中标记为 ``Start``).

机器人每次只能向下或者向右移动一步. 机器人试图达到网格的右下角 (在下图中标记为 ``Finish``).


.. figure:: ./robot_maze.png
   :align: center

   示意图

问总共有多少条不同的路径?

.. admonition:: 示例

    - 输入: ``m = 3``, ``n = 7``.
    - 输出: ``28``.

.. admonition:: 示例

    - 输入: ``m = 3``, ``n = 2``.
    - 输出: ``3``.
    - 解释: 从左上角开始, 总共有 3 条路径可以到达右下角.

      1. 向右 -> 向下 -> 向下.
      2. 向下 -> 向下 -> 向右.
      3. 向下 -> 向右 -> 向下.

.. admonition:: 示例

    - 输入: ``m = 7``, ``n = 3``.
    - 输出: ``28``.

.. admonition:: 示例

    - 输入: ``m = 3, n = 3``.
    - 输出: ``6``.

.. hint::

    - ``1 <= m, n <= 100``.
    - 题目数据保证答案小于等于 :math:`2 \cdot 10^9`.

题解
----

一共会走 :math:`m + n - 2` 步, 其中有 :math:`n-1` 步向下, :math:`m-1` 步向右. 所以一共有 :math:`C_{m+n-2}^{n-1} = C_{m+n-2}^{m-1}` 步.
